Task: Node recovery

1. Join your ucp-node0 and ucp-node1 nodes to the controller

2. Run three containers of your choice and take note of which nodes they are scheduled on

3. Between node0 and node1, SSH into the node that has more containers running

4. Stop the ucp-swarm-join container and wait a few minutes. Then refresh open the Nodes page in UCP. What do you notice?

5. Open the Containers page in UCP. What do you notice?

6. Start the ucp-swarm-join container and refresh your Nodes page to find the node again. Then check your containers on the node. Notice they are still running

7. Run a new container and set the restart policy on it to "always"

8. Take note of which node the container is on. SSH into that node and reboot it.
$ sudo reboot

9. Wait a few minutes until the node has rebooted.

10. Go into the containers page and check the containers on the node. Which container is still running after the reboot and which one has stopped?
